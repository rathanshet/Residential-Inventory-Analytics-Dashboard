Total SBA = SUM(inventory[SBA_SFT])

Avg Carpet Area = AVERAGE(inventory[CARPET_SFT])
Total Private Garden Area
Total Garden = SUM(inventory[PRIVATE_GARDEN_SFT])
🏗️ Inventory Analysis
Units by Tower
Units by Tower = 
CALCULATE(
  COUNTROWS(inventory),
  ALLEXCEPT(inventory, inventory[TOWER])
)
Average SBA by Flat Type
Avg SBA by Type = 
AVERAGEX(
  SUMMARIZE(inventory, inventory[CATEGORY]),
  CALCULATE(AVERAGE(inventory[SBA_SFT]))
)
Floor Distribution
Units per Floor = 
CALCULATE(
  COUNTROWS(inventory),
  ALLEXCEPT(inventory, inventory[FLOOR])
)
📐 Area Analysis
Carpet Area %
Carpet Ratio = 
DIVIDE(
  SUM(inventory[CARPET_SFT]),
  SUM(inventory[SBA_SFT])
) * 100
Balcony Contribution %
Balcony Ratio = 
DIVIDE(
  SUM(inventory[BALCONY_SFT]),
  SUM(inventory[SBA_SFT])
) * 100
Common Area %
Common Ratio = 
DIVIDE(
  SUM(inventory[COMMON_SFT]),
  SUM(inventory[SBA_SFT])
) * 100
🌿 Garden Analysis
Units with Private Garden
Garden Units = 
CALCULATE(
  COUNTROWS(inventory),
  inventory[HAS_GARDEN] = TRUE()
)
Average Garden per Unit
Avg Garden = 
CALCULATE(
  AVERAGE(inventory[PRIVATE_GARDEN_SFT]),
  inventory[HAS_GARDEN] = TRUE()
)
Garden Penetration %
Garden Penetration = 
DIVIDE(
  CALCULATE(COUNTROWS(inventory), inventory[HAS_GARDEN] = TRUE()),
  CALCULATE(COUNTROWS(inventory), inventory[IS_GROUND] = TRUE())
) * 100

Data Cleaning Pipeline

ETL process applied to the raw RERA Excel area statement

01
Raw Data Extraction
Read the "AREA statement" sheet from the Excel file using pandas.read_excel(). The first row is blank, row 2 contains column headers, and data begins from row 3.
02
Floor Column Forward-Fill
The FLOOR column is only populated at the start of each floor section. Applied ffill() to propagate floor values downward across all rows in that section.
data_df['FLOOR'] = data_df['FLOOR'].ffill()
03
Row Filtering
Dropped rows where the SL_NO column is null — these correspond to blank rows at the bottom of the sheet, not real units.
data_df = data_df[data_df['SL_NO'].notnull()]
04
Floor Normalization
Standardized raw floor values: "G " (with trailing space) → "G", and integer floors (1–10) left as strings.
def normalize_floor(f):
    f_str = str(f).strip()
    return 'G' if f_str.upper().startswith('G') else str(int(float(f_str)))
05
Private Garden Null Handling
The PRIVATE_GARDEN_SFT column is only populated for ground floor units. All upper-floor NaN values filled with 0 (no garden) instead of being left as missing.
data_df['PRIVATE_GARDEN_SFT'] = data_df['PRIVATE_GARDEN_SFT'].fillna(0)
06
Category Feature Engineering
The raw TYPE column contains values like "C 3B2T_TYPE 1-A". Extracted a simplified CATEGORY field (2B2T, 3B2T, 3B3T, 4B4T) and a BASE_TYPE stripping the -A/-B floor variant suffix.
data_df['CATEGORY'] = data_df['TYPE'].apply(get_category)
data_df['BASE_TYPE'] = data_df['TYPE'].str[:-2]
07
Derived Metrics
Computed ratio columns for Carpet, Balcony, Wall, and Common area as percentages of SBA. Added boolean flags IS_GROUND and HAS_GARDEN.
data_df['CARPET_RATIO'] = (data_df['CARPET_SFT'] / data_df['SBA_SFT'] * 100).round(2)
data_df['IS_GROUND'] = data_df['FLOOR'] == 'G'
data_df['HAS_GARDEN'] = data_df['PRIVATE_GARDEN_SFT'] > 0
08
SQLite Load + Views
Loaded the cleaned 272-row DataFrame into inventory.db and created four SQL views: v_tower_summary, v_floor_summary, v_category_summary, and v_garden_analysis.
data_df.to_sql('inventory', conn, if_exists='replace', index=False)
